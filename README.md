# 📖 [Awesome Books] <a name="about-project"></a>

**[Awesome Books]** is a simple website project that use plain Javascript and HTML to create a list of books on a page. It also implements a feature that removes a particular book selection

## 🛠 Built With <a name="built-with"></a>

- HTML
- Javascript
- CSS

- Browser
